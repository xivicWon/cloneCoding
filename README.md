## Step 2: Usage

```bash
   yarn add @react-native-community/netinfo
   // 추가 확인을 하기 위해 추가된 모듈
   yarn add react-native-toast-message
```

or npm

```bash
   npm install @react-native-community/netinfo
   // 추가 확인을 하기 위해 추가된 모듈
   npm install react-native-toast-message
```

### For iOS

```bash
   cd ios && pod install && cd ..
```
